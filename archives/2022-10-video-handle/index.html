视频下载处理工具合集

2022年10月25日 | you-get ffmpeg

You-Get 乃一小小哒命令行程序，提供便利的方式来下载网络上的媒体信息。

使用 conda 下载 you-get

https://anaconda.org/conda-forge/you-get

conda install -c conda-forge you-get

代理设置

使用 –http-proxy/-x为you-get设置HTTP代理:

$ you-get -x 127.0.0.1:7893 “URL”

使用 -l 下载列表文件:

$ you-get -l “URL”

使用 conda 下载 ffmpeg

https://anaconda.org/conda-forge/ffmpeg

conda install -c conda-forge ffmpeg

ffmpeg 常用命令（扩展中）

合并多文件

ffmpeg -f concat -safe 0 -i files.txt -vf "delogo=x=1080:y=14:w=185:h=65:show=0" -b:v 1M -c:a copy output.mp4 -y

剪切视频段

ffmpeg -ss 00:00:00.000 -t 15 -accurate_seek  -i -c:a copy output.mp4 -y

-vcodec copy -acodec copy 即-c:v copy -c:a copy 复制源文件的编码格式，转换会比较快。

设置视频的码率

-b:v 1M

设置视频的分辨率

-s 1920x1080

删除视频台标/水印/logo

多个以逗号分割
-vf "delogo=x=1440:y=40:w=260:h=40:show=0"

从视频中生成gif图片

ffmpeg -i input.mp4 -t 10 -s 320x240 -pix_fmt rgb24 output.gif
