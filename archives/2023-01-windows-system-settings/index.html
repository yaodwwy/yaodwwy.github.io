Windows 常用系统配置

2023年01月17日 | Windows

收录一些Windows常用的配置项。

系统变量表

变量名 描述 输出示例
%ALLUSERSPROFILE% 所有用户Profile文件位置 C:\ProgramData
%APPDATA% 应用程序数据的默认存放位置 C:\Users\Adam\AppData\Roaming
%CD% 当前目录 C:\Users\Adam
%CMDCMDLINE% 启动当前cmd.exe所使用的命令行 "C:\Windows\system32\cmd.exe"
%CMDEXTVERSION% 命令出当前命令处理程序扩展版本号 2
%CommonProgramFiles% 常用文件的文件夹路径 C:\ProgramFiles\CommonFiles
%COMPUTERNAME% 计算机名 ADAM-PC
%COMSPEC% 可执行命令外壳（命令处理程序）的路径 C:\Windows\system32\cmd.exe
%DATE% 当前日期 2023/01/17周二
%TIME% 当前时间 12:03:55.64
%ERRORLEVEL% 最近使用的命令的错误代码 0
%HOMEDRIVE% 与用户主目录所在的驱动器盘符 C
%HOMEPATH% 用户主目录的完整路径 \Users\Adam
%NUMBER_OF_PROCESSORS% 计算机安装的处理器数 4
%Path% 可执行文件的搜索路径 C:\Windows\system32;C:\Windows;...
%PATHEXT% 操作系统认为可被执行的文件扩展名 .COM;.EXE;.BAT;.CMD;.VBS;.VBE;.JS;.JSE;.WSF;.WSH;.MSC;.CPL
%PROCESSOR_ARCHITECTURE% 处理器的芯片架构 AMD64
%PROCESSOR_LEVEL% 计算机的处理器的型号 6
%PROCESSOR_REVISION% 处理器的修订号 9e09
%ProgramFiles% ProgramFiles文件夹的路径 C:\ProgramFiles
%PROMPT% 当前命令解释器的命令提示设置 $P$G
%RANDOM% 界于0和32767之间的随机十进制数 24894
%SYSTEMDRIVE% Windows启动目录驱动器 C
%SYSTEMROOT% Windows启动目录 C:\Windows
%TMP% 临时目录 C:\Users\Adam\AppData\Local\Temp
%USERDOMAIN% 包含用户帐号的域的名字 ADAM-PC
%USERNAME% 当前登录的用户的名字 Adam
%USERPROFILE% 当前用户Profile文件位置 C:\Users\Adam
%WINDIR% 操作系统目录的位置 C:\Windows

windows 端口转发

自Windows XP开始，Windows中就内置网络端口转发的功能。任何传入到本地端口的TCP连接（IPv4或IPv6）
都可以被重定向到另一个本地端口，或远程计算机上的端口，并且系统不需要有一个专门用于侦听该端口的服务。

netsh interface portproxy add v4tov4 listenaddress=10.0.3.123 listenport=22 connectaddress=192.168.56.1 connectport=22

依次为【windows ip】 【windows 端口】 【虚拟网卡ip】【虚拟网卡端口】

查看转发列表

netsh interface portproxy show  v4tov4

取消转发：

netsh interface portproxy delete v4tov4 listenaddress=【windows ip】  listenport=【windows 端口】
